Welcome to Fabrik!

With Fabrik, you can create your own models, using the large database of built-in layers. You also have the ability to use one of many built-in models. Another key feature of Fabrik is import and export of models to popular frameworks.



What is Fabrik?

Who created Fabrik?
Fabrik is an online collaborative platform to build and visualize deep learning models via a simple drag-and-drop interface. It allows researchers to collaboratively develop and debug models using a web GUI that supports importing, editing and exporting networks written in widely popular frameworks like Caffe, Keras, and TensorFlow.
 Fabrik was created by CloudCV - a young open source platform, aiming at making AI research more accessible. CloudCV was established in 2013 by students and faculty from the Machine Learning and Perception Lab at Virginia Tech. CloudCV's main goal is to make tools, which allow researchers and enthusiasts to build and compare machine learning models.
